# Stock & Finance AI Website - Quickstart

## Getting Started
1. `cd frontend && npm install`
2. `cd backend && npm install`
3. `cd ai-models && python -m venv venv && .\venv\Scripts\activate && pip install -r requirements.txt`

## Running
- Terminal 1: `cd frontend && npm run dev`
- Terminal 2: `cd backend && npm run dev`
- Terminal 3: `cd ai-models && .\venv\Scripts\activate && python app.py`
